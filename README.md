# MasterCare Lucky Number Wheel

A responsive static lucky wheel for MasterCare.

## Numbers

The wheel contains these 22 unique numbers:

`8, 16, 4, 45, 11, 40, 10, 1, 12, 9, 50, 51, 52, 43, 20, 22, 54, 44, 24, 56, 70, 71`

## Spins

Visitors can spin the wheel multiple times. The popup result is calculated from the final number positioned under the arrow.
